== Description ==

<h1>NoMoreIE6</h1>
This Word Press Plugin adds a browser dependent upgrade-message to the website header, explaining the risks of using IE6 and providing links with icons to upgrade to a recent browser.

It only uses CSS conditions, no scripts. It's totally unobtrusive, and built with valid W3C code.

== Installation ==

1. Upload `nomoreie6.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where does the warning message shows? =

The warning message will show on on the top of your website, just before the website's header

= Will it cover the my website's header? =

It will not cover any part of your website, since it will push it a few pixels down to accomodate the warning

= Does it uses Scripts that can comprimise my website? =

No it doesn't use any script, it only uses conditional CSS to display the warning message

= is the code W3C valid? =

The code is w3C valid and passes W3C automatic Validation for XHTML 1.0 Transitional
